# v2.3 Features Design

Date: 2026-03-18
Scope: 5 tasks — 3 UX + 2 content sections
Strategy: parallel implementation, single commit at the end

## UX-1: Copy-as-Markdown (Quick Scan)

- Add `Copy as MD` button in `#quick-scan .section-header`
- On click: generate markdown table from 14 `.red-flag-row` elements
- Format: `| Pattern | Description |` table
- Use `navigator.clipboard.writeText()` + 2s "Copied!" toast
- Button style: `.btn-ghost` variant matching existing UI

## UX-2: `?` Keyboard Shortcuts Overlay

- New `#shortcuts-modal` element (hidden by default) appended to `<body>`
- Lists: `/` focus search, `Escape` clear/close, `?` toggle overlay
- Backdrop: semi-transparent overlay, click-to-close
- Toggle via `keydown` listener on `?` key (skip if in input/textarea)
- CSS: centered modal, table layout for shortcut rows

## UX-3: Filter State in URL Hash Params

- Params: `?s=Critical&t=DeFi` (severity + type filters)
- On filter change: `history.replaceState()` updates URL
- On page load: `URLSearchParams` reads state before first `applyFilters()`
- Handles partial state (e.g., only `?s=High` without type)

## Content-4: Storage Layout / ERC-7201 Section

- New section `#storage-layout` after `#crosschain-security`
- One vuln card `#storage-collision` (High, Infrastructure type)
- Covers:
  - Unstructured storage slot pattern (pre-ERC-7201)
  - Storage layout corruption in upgrades (add/remove variables)
  - ERC-7201 namespaced storage with `@custom:storage-location`
  - `keccak256(abi.encode(uint256(keccak256("namespace")) - 1)) & ~bytes32(uint256(0xff))`
- Code blocks: vulnerable upgrade pattern + safe ERC-7201 pattern
- Sidebar: add link in "Advanced Topics" nav

## Content-5: Vyper Vulnerabilities Section

- New section `#vyper-security` at end of Advanced Topics
- One card `#vyper-vulns` (High, Infrastructure type)
- Covers:
  - Reentrancy in versions <=0.2.15, 0.3.0, 0.3.1 (Curve $73M exploit, Aug 2023)
  - Bounds check bypass in storage arrays (pre-0.3.10)
  - Default visibility misconception (all funcs external by default)
- Code blocks: vulnerable (Vyper 0.2.x) vs safe (0.3.10+) per topic
- Sidebar: add link under "Advanced Topics" nav
